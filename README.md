# House Price Prediction System

A machine learning project developed using Python and Scikit-learn to predict house prices based on various property-related features. The project focuses on data preprocessing, exploratory data analysis, model training, and prediction using regression techniques.

---

## Project Overview

This project uses machine learning algorithms to estimate house prices from housing datasets. The workflow includes:

* Data collection and preprocessing
* Exploratory Data Analysis (EDA)
* Feature selection
* Model training and evaluation
* House price prediction
* Data visualization using graphs and plots

The project was developed and executed using Google Colab.
